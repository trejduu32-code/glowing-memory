# Cron Job Service - Unlimited Edition

A full-featured cron job service with NO LIMITS on usage.

## Features

- **NO RATE LIMITS**: Unlimited executions per minute
- **NO CONCURRENCY LIMITS**: Jobs can run simultaneously without restrictions
- **UNLIMITED JOBS**: Create as many cron jobs as you need
- User authentication (register/login/logout)
- Create, manage, and delete cron jobs
- View execution logs for each job
- Server-side rendering with EJS
- SQLite database (no external dependencies)
- Ready for Render deployment
